# 🌊 ТвойМаяк — AI-дневник

Личный дневник с ИИ. Лента с инсайтами, трекинг настроения,
поддержка в сложные дни, Liquid Glass дизайн.

---

## Быстрый старт (3 шага)

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Запусти сервер
```bash
python app.py
```

### 3. Открой браузер
```
http://localhost:5000
```

---

## Структура проекта
```
tvoymaak/
├── app.py              # Flask-сервер + API + ИИ-логика
├── requirements.txt    # Зависимости
├── tvoymaak.db         # SQLite база (создаётся автоматически)
├── templates/
│   ├── index.html      # Страница входа/регистрации
│   └── app.html        # Главное приложение (SPA)
└── static/             # Статика (если понадобится)
```

## Функционал

### Лента
- Прошлые записи с фото
- AI-саммари каждого дня
- Инсайт недели от ИИ
- Трекер серии (streak)

### Твой день
- Оценка настроения (5 уровней)
- Чекбоксы задач (можно добавлять свои)
- Описание дня
- Фото дня
- AI-саммари + рекомендации после сохранения
- При плохом настроении — кнопка чата с ИИ

### Поддержка
- Живой чат с Маяком (ИИ)
- Техники дыхания при плохом настроении
- Эмпатичные ответы

## Деплой на сервер

```bash
# Установи gunicorn
pip install gunicorn

# Запуск через gunicorn
gunicorn -w 2 -b 0.0.0.0:5000 app:app

# Или с systemd-сервисом для автозапуска
```

Секретный ключ для продакшна:
```bash
export SECRET_KEY="your-very-secret-random-string"
python app.py
```
