# 🌊 ТвойМаяк — AI-дневник

**Проект, созданный в рамках Хакатона по ИИ при поддержке Яндекс Образования**

Личный дневник с ИИ. Лента с инсайтами, трекинг настроения,
поддержка в сложные дни, современный Liquid Glass дизайн.



В современном мире мы часто забываем о своих чувствах, теряем связь с собой. ТвойМаяк помогает её восстановить: вы записываете свои мысли, отмечаете настроение, а ИИ бережно анализирует текст, выделяет ключевые моменты и даёт мягкие рекомендации. Если вам грустно - Маяк всегда рядом: предложит дыхательную практику или просто выслушает в чате.

---

## Быстрый старт (3 шага)

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Создайте файл .env в корне проекта:
```
mkdir .env
```

```
YANDEX_API_KEY=...
YANDEX_BASE_URL=https://...
YANDEX_PROJECT=...
```


### 3. Запусти сервер
```bash
python app.py
```

### 4. Открой браузер
```
http://localhost:5000
```

## Функционал

### Лента
- Прошлые записи с фото
- AI-саммари каждого дня
- Инсайт недели от ИИ
- Трекер серии (streak)

### Твой день
- Оценка настроения (5 уровней)
- Чекбоксы задач (можно добавлять свои)
- Описание дня
- Фото дня
- AI-саммари + рекомендации после сохранения
- При плохом настроении — кнопка чата с ИИ

### Поддержка
- "Живой" чат с Маяком (ИИ)
- Помощь при плохом настроении

## Дизайн и пользовательский опыт
- Liquid Glass - имитация текучего стекла: полупрозрачные панели, размытие фона, мягкие тени, плавные переходы.
- Адаптивность - интерфейс удобен и на десктопе, и на мобильном телефоне (мобильная версия с вертикальной навигацией).
- Тёмная тема - по умолчанию, снижает нагрузку на глаза.
- Микроанимации - при наведении, нажатии кнопок, появлении модальных окон.
- Интуитивная навигация - три основных экрана:
- Лента - хронология записей.
- Твой день - форма для создания/редактирования записи.
- Поддержка - чат и дыхательные упражнения.

## Деплой на сервер

```bash
# Установи gunicorn
pip install gunicorn

# Запуск через gunicorn
gunicorn -w 2 -b 0.0.0.0:5000 app:app

# Или с systemd-сервисом для автозапуска
```

Секретный ключ для продакшна:
```bash
export SECRET_KEY="your-very-secret-random-string"
python app.py
```
